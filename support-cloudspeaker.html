同梱物の確認

箱を開けたら、以下のものが入っているか確認してください。

  • スピーカー本体 ... 2台（左・右）
  • USB ケーブル ... 1本（電源＆音声入力用、PVC ホワイト）
  • スピーカー接続ケーブル ... 1本（左右のスピーカーをつなぐ用、3.5mm 4極）
  • 保証書・かんたんガイド ... 1枚
足りないものがあったら
お手数ですが、下記のお問い合わせ先までご連絡ください。すぐに対応いたします。
2

接続方法

かんたん3ステップで、すぐに使い始められます。

左右のスピーカーをつなぐ

付属のスピーカー接続ケーブル（3.5mm端子）で、2台のスピーカーを接続します。ケーブルの端子は同じ形なので、どちらに挿してもOKです。

USBケーブルをパソコンに接続

付属のUSBケーブルを、メイン側のスピーカー（ボタンがある方）とパソコンのUSBポートに接続します。電源も音声もこの1本でOK。ドライバーのインストールは不要です。

音を鳴らしてみよう

パソコンの音声出力先を「MNC-CL01」に設定すれば準備完了。お気に入りの音楽を再生してみてください。

おすすめ
USB 3.2 以上のポート、または独立給電のUSBハブに接続すると、安定した電力供給で最高の音質をお楽しみいただけます。動作電圧は 4.75 ~ 5.25V です。

取り扱い上の注意

安全にお使いいただくために
以下の点にご注意ください。
  • 水濡れに注意
    本製品は防水仕様ではありません。飲み物のそばに置く際はご注意ください。
  • 直射日光・高温を避ける
    直射日光が当たる場所や、暖房器具のそばなど高温になる場所に置かないでください。（使用温度範囲: -20℃ ~ 50℃）
  • ケーブルを無理に曲げない
    ケーブルを鋭く折り曲げたり、強く引っ張ったりしないでください。引張強度は3kg以上ありますが、断線の原因になります。
  • 落下にご注意
    高い場所からの落下は故障の原因になります。安定した平面の上でお使いください。
  • 分解・改造しない
    内部に触れると故障や感電の原因になります。修理が必要な場合はお問い合わせください。
  • お手入れ方法
    乾いた柔らかい布で拭いてください。アルコールやシンナーなどの溶剤は使用しないでください。表面のラバー塗装が傷む場合があります。

環境への配慮

本製品はRoHS（有害物質使用制限）基準に適合しています。廃棄の際はお住まいの自治体のルールに従ってください。

よくある質問

Q ドライバーのインストールは必要ですか？ +
いいえ、不要です。USBを接続するだけで、Windows / Mac / Linux で自動認識されます。ドライバーレスですぐにお使いいただけます。
Q 音が出ません +
以下を確認してください:
1. パソコンの音声出力先が「MNC-CL01」になっていますか？
2. 音量がミュート（消音）になっていませんか？
3. USBケーブルがしっかり接続されていますか？
4. 別のUSBポートに挿し替えてみてください。USB 3.2以上のポートがおすすめです。
それでも解決しない場合は、お問い合わせください。
Q 左右の音量が違います +
パソコンのサウンド設定で、左右のバランスが均等になっているか確認してください。また、左右のスピーカー接続ケーブル（3.5mm端子）がしっかり差し込まれているかもご確認ください。
Q スマートフォンでも使えますか？ +
USB-C対応のスマートフォンであれば、USB OTGアダプター（別売）を介して使用できる場合があります。ただし、すべてのスマートフォンでの動作を保証するものではありません。
Q LEDだけ消すことはできますか？ +
はい、できます。ボタンを長押しするとLEDのオン/オフを切り替えられます。LEDを消しても音声出力には影響ありません。
Q 音割れ・雑音がします +
音量を下げてみてください。最大音量での使用は定格電力（3W）を超える場合があります。また、USBポートの電力供給が不安定な場合にも発生することがあります。独立給電のUSBハブを試してみてください。
Q USBハブ経由でも使えますか？ +
はい、使えます。ただし、電力不足による音質低下を防ぐため、セルフパワー（電源アダプター付き）のUSBハブをおすすめします。最大消費電力は7Wです。
