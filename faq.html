ご購入について— Purchase

Q.お支払い方法は何がありますか？
クレジットカード（Visa / Mastercard / JCB / American Express）、PayPay、銀行振込、コンビニ決済に対応しております。手数料はお客様のご負担となります。
Q.ギフトラッピングは有料ですか？
minotel公式サイトでご購入いただいた商品は、すべて無料でギフトラッピングしてお届けします。メッセージカードの添付もご注文時にご指定いただければ無料でお付けできます。
Q.複数個まとめて購入できますか？
はい、複数個ご注文いただけます。法人・学校・イベント向けなど、10個以上のまとめ買いをご検討の方は、お問い合わせフォームよりご相談ください。
Q.領収書は発行してもらえますか？
はい、ご注文時の備考欄、もしくは注文後のメールにて、宛名・但し書きをお知らせください。PDFにてお送りいたします。
02

配送について— Shipping

Q.送料はいくらかかりますか？
本州・四国・九州・北海道は送料無料、沖縄・離島は800円（税込）です。詳細は配送ポリシーをご確認ください。
Q.ご注文後、どのくらいで届きますか？
ご注文確認後、2〜5営業日以内に発送いたします。発送後は、地域によりますが1〜3日程度でお届けとなります。離島・一部地域は通常より日数がかかる場合があります。
Q.配送業者はどこですか？
佐川急便またはヤマト運輸にて配送いたします。配送業者のご指定はお受けしておりませんので、あらかじめご了承ください。
Q.海外への発送は可能ですか？
現在、海外への発送には対応しておりません。国内のみの対応となります。今後の対応については検討中です。
Q.配送日時の指定はできますか？
ご注文時の備考欄にご希望の配送日・時間帯をご記入ください。できる限り対応いたしますが、ご注文混雑時はご希望に添えない場合があります。
03

商品・使い方について— Product

Q.ドライバーのインストールは必要ですか？
いいえ、不要です。USBを接続するだけで、Windows / Mac / Linux で自動認識されます。ドライバーレスですぐにお使いいただけます。
Q.スマートフォンでも使えますか？
USB-C対応のスマートフォンであれば、USB OTGアダプター（別売）を介して使用できる場合があります。ただし、すべての機種での動作を保証するものではありません。
Q.LEDだけ消すことはできますか？
はい、できます。ボタンを長押しするとLEDのオン／オフを切り替えられます。LEDを消しても音声出力には影響ありません。
Q.BluetoothやWi-Fiには対応していますか？
いいえ、本製品はUSB接続専用です。BluetoothやWi-Fiには対応しておりません。USB接続のため、遅延なく安定した音質でお使いいただけます。
Q.USBハブ経由でも使えますか？
はい、お使いいただけます。ただし電力不足による音質低下を防ぐため、セルフパワー（電源アダプター付き）のUSBハブをおすすめします。最大消費電力は7Wです。
04

トラブルシューティング— Trouble

Q.音が出ません
以下をご確認ください。
1. パソコンの音声出力先が「MNC-CL01」になっているか
2. 音量がミュート（消音）になっていないか
3. USBケーブルがしっかり接続されているか
4. 別のUSBポートに挿し替えてみる
解決しない場合はお問い合わせください。
Q.左右の音量が違います
パソコンのサウンド設定で左右のバランスが均等になっているかご確認ください。左右のスピーカー接続ケーブル（3.5mm端子）がしっかり差し込まれているかもご確認ください。
Q.音割れ・雑音がします
音量を下げてみてください。最大音量での使用は定格電力（3W）を超える場合があります。また、USBポートの電力供給が不安定な場合にも発生することがあるので、別のポートまたは独立給電のUSBハブをお試しください。
Q.LEDが点灯しません
ボタン長押しでLEDがオフになっている可能性があります。再度ボタンを長押ししてオンにしてみてください。それでも点灯しない場合はお問い合わせください。

返品・交換について— Return

Q.返品・交換はできますか？
初期不良や商品の瑕疵の場合は、商品到着後14日以内であれば送料当ブランド負担で返品・交換を承ります（開封後でも対象）。お客様都合の返品は、商品到着後8日以内かつ未開封の場合のみお受けします。詳細は返金ポリシーをご確認ください。
Q.開封後に汚れや傷を見つけました
初期不良として返品・交換の対象となります。開封後でも対応いたしますので、ご安心ください。商品到着後14日以内に、お問い合わせより該当箇所の写真を添えてご連絡ください。
Q.メーカー保証はありますか？
ご購入日から12ヶ月間のメーカー保証が付属しております。通常使用において故障が発生した場合は、保証期間内であれば無償修理または交換にて対応いたします。
Q.保証対象外になるのはどんな場合ですか？
お客様の過失による破損、改造・分解、水濡れ、落下による故障、通常使用範囲を超える使用による劣化などは保証対象外となります。
